# KOPPSUISSE Inventur Android-App / PWA

Installierbare Web-App für Samsung/Android.

Regel:
- 1 Odoo-Einheit = 1 Paar L+R
- 1 Karton = 10 Paar = 20 einzelne Schienen
- 25 Kartons = 250 Paar

Installation als App auf Samsung:
1. Ordner auf eine HTTPS-Webseite hochladen, z. B. shop.koppsuisse.ch/inventur-app/
2. Auf Samsung mit Chrome oder Samsung Internet öffnen
3. Menü ⋮ öffnen
4. "App installieren" oder "Zum Startbildschirm hinzufügen" drücken

Direkt aus dem Dateimanager geöffnet läuft sie als normale Webseite, aber nicht immer als echte installierte App.
