Installation auf Samsung

Wenn diese App über eine Webseite/HTTPS geöffnet wird, kann sie wie eine App installiert werden:

Chrome oder Samsung Internet → Menü ⋮ → App installieren oder Zum Startbildschirm hinzufügen.

Wenn du diese Datei nur direkt öffnest, läuft sie als normale Webseite. Für echte App-Installation muss sie auf eine Webseite hochgeladen werden.
