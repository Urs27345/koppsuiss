# Salut Urs - PWA

Einfacher Pfad: /gesund/

## Hochladen
Den Ordner `gesund` direkt in den öffentlichen Web-Ordner hochladen.

Danach öffnen:
https://www.koppsuisse.ch/gesund/

Auf der Startseite erscheint nur der Button `App installieren`.

## Dateien im Ordner gesund
- index.html
- manifest.webmanifest
- service-worker.js
- icon-192.png
- icon-512.png

Wichtig: Alle Dateien müssen direkt in `gesund` liegen, nicht in einem weiteren Unterordner.
