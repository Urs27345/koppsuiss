# Salud Urs Blutdruck-App

Neuaufbau für Android APK.

Funktionen:
- Blutdruckwerte manuell speichern
- Foto aufnehmen und sicher in der App-Vorschau anzeigen
- CSV exportieren/teilen
- Lokaler Speicher auf dem Android-Handy

Upload-Ziel:
Diese Dateien direkt in `public/blut/` entpacken.
Der `.github` Ordner bleibt unverändert.
